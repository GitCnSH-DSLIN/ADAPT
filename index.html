Welcome to the ADAPT Project.

This project replaces the "LKSL" library, providing constituents of a powerful development framework for a broad range of high-performance, multi-threaded systems.

ADAPT is largely built up of constituents from a powerful game engine project called "AGE" (Another Game Engine), which have been abstracted from the game engine for use in an infinite number of potential applications. These can range from something as simple as a time-critical automation tool, to complete workflow management systems, data-driven systems, or scientific applications.
